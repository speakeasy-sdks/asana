# AuditLogEventContextContextType

The type of context.
Can be one of `web`, `desktop`, `mobile`, `asana_support`, `asana`, `email`, or `api`.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `Web`          | web            |
| `Desktop`      | desktop        |
| `Mobile`       | mobile         |
| `AsanaSupport` | asana_support  |
| `Asana`        | asana          |
| `Email`        | email          |
| `Api`          | api            |
