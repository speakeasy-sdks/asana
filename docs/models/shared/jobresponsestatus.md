# JobResponseStatus

The current status of this job. The value is one of: `not_started`, `in_progress`, `succeeded`, or `failed`.


## Values

| Name         | Value        |
| ------------ | ------------ |
| `NotStarted` | not_started  |
| `InProgress` | in_progress  |
| `Succeeded`  | succeeded    |
| `Failed`     | failed       |
